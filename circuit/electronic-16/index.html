Lesson 16 基本运算电路

约 1031 字大约 3 分钟

2025-12-26

集成运放引入负反馈，工作在线性区，利用“虚短”、“虚断”分析问题是基本出发点，结合电路理论中的基本知识，如KCL、KVL、VCR、叠加原理等；

本章给出的典型的运算电路都是电压型运算电路，即输出电压是输入电压的函数，稳定输出电压；事实上运算电路不局限于电压运算电路，掌握本章几种典型运算电路的分析方法，对其他的由集成运放引入负反馈、工作在线性区的电路的分析同样适用；

· 比例运算电路

反相比例运算电路：

对 N 点应用虚断与KCL：

uI−uNR=uN−uORf\frac{u_I - u_N}{R} = \frac{u_N - u_O}{R_f} RuI​−uN​​=Rf​uN​−uO​​

虚短：

由于虚断，R′R'R′ 上无压降，故

uN=uP=0u_N = u_P = 0 uN​=uP​=0

代入得输出电压：

uO=−RfRuIu_O = -\frac{R_f}{R} u_I uO​=−RRf​​uI​

虽然电阻 R′R'R′ 在运算关系式中没有出现，但一般为了保证两个输入端口等效电阻的对称性，通常选择令 RP=RNR_P = R_NRP​=RN​，即：

R′=R∥RfR' = R \parallel R_f R′=R∥Rf​

T型反相比例运算电路：

对 M 点应用 KCL：

uN−uMR2+0−uMR3=uM−uOR4\frac{u_N - u_M}{R_2} + \frac{0 - u_M}{R_3} = \frac{u_M - u_O}{R_4} R2​uN​−uM​​+R3​0−uM​​=R4​uM​−uO​​

对 N 点应用虚断和 KCL：

uI−uNR1=uN−uMR2\frac{u_I - u_N}{R_1} = \frac{u_N - u_M}{R_2} R1​uI​−uN​​=R2​uN​−uM​​

虚短：

由于虚断，R′R'R′ 上无压降，故有

uN=uP=0u_N = u_P = 0 uN​=uP​=0

推导得输出电压表达式：

uo=−R2R4R3⋅1R1+R2+R4+R2R4R3⋅uIu_o = -\frac{R_2 R_4}{R_3} \cdot \frac{1}{R_1 + R_2 + R_4 + \frac{R_2 R_4}{R_3}} \cdot u_I uo​=−R3​R2​R4​​⋅R1​+R2​+R4​+R3​R2​R4​​1​⋅uI​

T 型反相比例运算器有什么优点？可实现较大的增益，同时避免使用过大阻值的电阻

同相比例运算电路：

对 N 点应用虚断与KCL：

0−uNR=uN−uORf\frac{0 - u_N}{R} = \frac{u_N - u_O}{R_f} R0−uN​​=Rf​uN​−uO​​

虚短：

uN=uPu_N = u_P uN​=uP​

P 点虚断，$ R' $ 上压降为零：

uP=uIu_P = u_I uP​=uI​

联立得输出电压公式：

uO=(1+RfR)uIu_O = \left(1 + \frac{R_f}{R}\right) u_I uO​=(1+RRf​​)uI​

电压跟随器：

当同相比例运算器中的 Rf=0R_f = 0Rf​=0（即短路），或 R=∞R = \inftyR=∞（即开路）时，输出电压等于输入电压，称为电压跟随器.

由于其具有：输入电阻无穷大，输出电阻为零.

因此可以用于前后级隔离，实现信号缓冲作用。

同相比例运算电路一般形式：

电路分析：对 N 点和 P 点应用虚断与 KCL

对 N 点应用虚断、KCL：

0−uNR1=uN−uORf\frac{0 - u_N}{R_1} = \frac{u_N - u_O}{R_f} R1​0−uN​​=Rf​uN​−uO​​

虚短：

uN=uPu_N = u_P uN​=uP​

对 P 点应用虚断、KCL：

uI−uPR2=uP−0R3\frac{u_I - u_P}{R_2} = \frac{u_P - 0}{R_3} R2​uI​−uP​​=R3​uP​−0​

推导得输出电压表达式：

uO=RpRN⋅RfR2⋅u1u_O = \frac{R_p}{R_N} \cdot \frac{R_f}{R_2} \cdot u_1 uO​=RN​Rp​​⋅R2​Rf​​⋅u1​

其中：

Rp=R2∥R3R_p = R_2 \parallel R_3 Rp​=R2​∥R3​

RN=R1∥RfR_N = R_1 \parallel R_f RN​=R1​∥Rf​

· 加减运算电路

反相求和运算电路：

对 N 点应用虚断、KCL：

ui1−uNR1+ui2−uNR2+ui3−uNR3=uN−uORf\frac{u_{i1} - u_N}{R_1} + \frac{u_{i2} - u_N}{R_2} + \frac{u_{i3} - u_N}{R_3} = \frac{u_N - u_O}{R_f} R1​ui1​−uN​​+R2​ui2​−uN​​+R3​ui3​−uN​​=Rf​uN​−uO​​

虚短：（由于虚断 R' 上无压降）

uN=uP=0u_N = u_P = 0 uN​=uP​=0

uO=−(RfR1ui1+RfR2ui2+⋯+RfRnuin)u_O = -(\frac{R_f}{R_1}u_{i1} + \frac{R_f}{R_2}u_{i2} + \cdots + \frac{R_f}{R_n}u_{in}) uO​=−(R1​Rf​​ui1​+R2​Rf​​ui2​+⋯+Rn​Rf​​uin​)

同相求和运算电路：

对 N 点应用虚断、KCL：

0−uNR=uN−uORf\frac{0 - u_N}{R} = \frac{u_N - u_O}{R_f} R0−uN​​=Rf​uN​−uO​​

虚短：

uN=uPu_N = u_P uN​=uP​

对 P 点应用虚断、KCL：

ui1−uPR1+ui2−uPR2+ui3−uPR3=uP−0R4\frac{u_{i1} - u_P}{R_1} + \frac{u_{i2} - u_P}{R_2} + \frac{u_{i3} - u_P}{R_3} = \frac{u_P - 0}{R_4} R1​ui1​−uP​​+R2​ui2​−uP​​+R3​ui3​−uP​​=R4​uP​−0​

uO=RpRN(RfR1ui1+RfR2ui2+⋯+RfRnuin)u_O = \frac{R_p}{R_N}(\frac{R_f}{R_1}u_{i1} + \frac{R_f}{R_2}u_{i2} + \cdots + \frac{R_f}{R_n}u_{in}) uO​=RN​Rp​​(R1​Rf​​ui1​+R2​Rf​​ui2​+⋯+Rn​Rf​​uin​)

加减运算电路：

利用叠加原理：

uO=RpRN(RfR3ui3+RfR4ui4)−(RfR1ui1+RfR2ui2)u_O = \frac{R_p}{R_N}(\frac{R_f}{R_3}u_{i3} + \frac{R_f}{R_4}u_{i4}) - (\frac{R_f}{R_1}u_{i1} + \frac{R_f}{R_2}u_{i2}) uO​=RN​Rp​​(R3​Rf​​ui3​+R4​Rf​​ui4​)−(R1​Rf​​ui1​+R2​Rf​​ui2​)

当 $ R_p = R_N $ 时：

uO=Rf(−ui1R1−ui2R2+ui3R3+ui4R4)u_O = R_f(-\frac{u_{i1}}{R_1} - \frac{u_{i2}}{R_2} + \frac{u_{i3}}{R_3} + \frac{u_{i4}}{R_4}) uO​=Rf​(−R1​ui1​​−R2​ui2​​+R3​ui3​​+R4​ui4​​)

同相部分乘以 RpRN\frac{R_p}{R_N}RN​Rp​​ ，反相部分不乘！
