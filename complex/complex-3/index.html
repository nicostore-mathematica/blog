· 复级数

我们先给出复级数的定义.

/Definition/

形如 f1+f2+⋯+fk+⋯=∑k=1∞fkf_1 + f_2 + \cdots + f_k + \cdots = \displaystyle\sum_{k=1}^{\infty} f_kf1​+f2​+⋯+fk​+⋯=k=1∑∞​fk​ 的级数（其中 fk=uk+ivk∈Cf_k = u_k + i v_k \in \mathbb{C}fk​=uk​+ivk​∈C）称为复数项级数。显然，我们有

∑k=1∞fk=∑k=1∞uk+i∑k=1∞vk.\sum_{k=1}^{\infty} f_k = \sum_{k=1}^{\infty} u_k + i \sum_{k=1}^{\infty} v_k. k=1∑∞​fk​=k=1∑∞​uk​+ik=1∑∞​vk​.

部分和定义为

Fn≡∑k=1nfk,F_n \equiv \sum_{k=1}^{n} f_k, Fn​≡k=1∑n​fk​,

可构成部分和序列 {Fn}\{F_n\}{Fn​}。

若 {Fn}\{F_n\}{Fn​} 收敛，即

lim⁡n→∞Fn=F\lim_{n \to \infty} F_n = F n→∞lim​Fn​=F

存在且有限，则称 ∑k=1∞fk\displaystyle\sum_{k=1}^{\infty} f_kk=1∑∞​fk​ 收敛于 F=u+ivF = u + i vF=u+iv。即 ∑k=1∞fk\displaystyle\sum_{k=1}^{\infty} f_kk=1∑∞​fk​ 收敛等价于

∑k=1∞uk=u,∑k=1∞vk=v.\sum_{k=1}^{\infty} u_k = u, \quad \sum_{k=1}^{\infty} v_k = v. k=1∑∞​uk​=u,k=1∑∞​vk​=v.

因此，复级数收敛问题可以转化为实级数收敛问题。

由于 ∑n=0∞∣un∣\displaystyle\sum_{n=0}^{\infty} |u_n|n=0∑∞​∣un​∣ 是实数级数，而且是正项级数，所以高等数学中的任何一种正项级数的收敛判别法都可以用来判别复数级数是否绝对收敛。下面列出最常用的几个判别法。

比较判别法

若 ∃N∈N\exists N \in \mathbb{N}∃N∈N，对 ∀n>N\forall n > N∀n>N，都有 ∣un∣<vn|u_n| < v_n∣un​∣<vn​，而 ∑n=0∞vn\displaystyle\sum_{n=0}^{\infty} v_nn=0∑∞​vn​ 收敛，则 ∑n=0∞∣un∣\displaystyle\sum_{n=0}^{\infty} |u_n|n=0∑∞​∣un​∣ 收敛，即 ∑n=0∞un\displaystyle\sum_{n=0}^{\infty} u_nn=0∑∞​un​ 绝对收敛。

若 ∣un∣>vn>0|u_n| > v_n > 0∣un​∣>vn​>0，而 ∑n=0∞vn\displaystyle\sum_{n=0}^{\infty} v_nn=0∑∞​vn​ 发散，则 ∑n=0∞∣un∣\displaystyle\sum_{n=0}^{\infty} |u_n|n=0∑∞​∣un​∣ 发散。

若存在与 nnn 无关的常数 ρ\rhoρ，则当 ∣un+1/un∣<ρ<1|u_{n+1}/u_n| < \rho < 1∣un+1​/un​∣<ρ<1 时，级数 ∑n=0∞un\displaystyle\sum_{n=0}^{\infty} u_nn=0∑∞​un​ 绝对收敛；

当 ∣un+1/un∣>ρ>1|u_{n+1}/u_n| > \rho > 1∣un+1​/un​∣>ρ>1 时，级数 ∑n=0∞un\displaystyle\sum_{n=0}^{\infty} u_nn=0∑∞​un​ 发散。

d'Alembert 判别法

若 lim‾⁡n→∞∣un+1/un∣<1\displaystyle\varlimsup_{n \to \infty} |u_{n+1}/u_n| < 1n→∞lim​∣un+1​/un​∣<1，则级数 ∑n=0∞un\displaystyle\displaystyle\sum_{n=0}^{\infty} u_nn=0∑∞​un​ 绝对收敛；若 lim‾⁡n→∞∣un+1/un∣>1\displaystyle\varliminf_{n \to \infty} |u_{n+1}/u_n| > 1n→∞lim​​∣un+1​/un​∣>1，则级数 ∑n=0∞un\displaystyle\sum_{n=0}^{\infty} u_nn=0∑∞​un​ 发散。

若 lim⁡n→∞∣un+1/un∣=1\displaystyle\displaystyle\lim_{n \to \infty} |u_{n+1}/u_n| = 1n→∞lim​∣un+1​/un​∣=1，则 ∑n=0∞un\displaystyle\displaystyle\sum_{n=0}^{\infty} u_nn=0∑∞​un​ 的绝对收敛性需要利用下面的 Gauss 判别法进一步检验。

Gauss 判别法

设级数 ∑n=0∞un\displaystyle\sum_{n=0}^{\infty} u_nn=0∑∞​un​ 邻项的比值可以写成

unun+1=1+μn+O(n−λ),\frac{u_n}{u_{n+1}} = 1 + \frac{\mu}{n} + O(n^{-\lambda}), un+1​un​​=1+nμ​+O(n−λ),

其中 μ=a+ib\mu = a + ibμ=a+ib，λ>1\lambda > 1λ>1，符号 OOO 表示数量级。若 a>1a > 1a>1，则级数 ∑n=0∞un\displaystyle\sum_{n=0}^{\infty} u_nn=0∑∞​un​ 绝对收敛；若 a≤1a \leq 1a≤1，则 ∑n=0∞∣un∣\displaystyle\sum_{n=0}^{\infty} |u_n|n=0∑∞​∣un​∣ 发散。

Cauchy 判别法

若 lim‾⁡n→∞∣un∣1/n<1\varlimsup_{n \to \infty} |u_n|^{1/n} < 1limn→∞​∣un​∣1/n<1，则级数 ∑n=0∞un\displaystyle\sum_{n=0}^{\infty} u_nn=0∑∞​un​ 绝对收敛；若 lim‾⁡n→∞∣un∣1/n>1\varliminf_{n \to \infty} |u_n|^{1/n} > 1lim​n→∞​∣un​∣1/n>1，则级数 ∑n=0∞un\displaystyle\sum_{n=0}^{\infty} u_nn=0∑∞​un​ 发散。

绝对收敛级数具有下列性质：

(1) 改换次序。例如，

u0+u1+u2+u3+u4+⋯=u0+u1+u2+u4+u3+u6+u8+u5+⋯ .u_0 + u_1 + u_2 + u_3 + u_4 + \cdots = u_0 + u_1 + u_2 + u_4 + u_3 + u_6 + u_8 + u_5 + \cdots. u0​+u1​+u2​+u3​+u4​+⋯=u0​+u1​+u2​+u4​+u3​+u6​+u8​+u5​+⋯.

(2) 特别是，可以把绝对收敛级数拆成几个子级数，每个子级数仍绝对收敛。例如，

∑n=0∞un=∑n=0∞u2n+∑n=0∞u2n+1.\sum_{n=0}^{\infty} u_n = \sum_{n=0}^{\infty} u_{2n} + \sum_{n=0}^{\infty} u_{2n+1}. n=0∑∞​un​=n=0∑∞​u2n​+n=0∑∞​u2n+1​.

(3) 两个绝对收敛级数之积仍然绝对收敛，

∑kuk⋅∑lvl=∑k,lukvl.\sum_k u_k \cdot \sum_l v_l = \sum_{k,l} u_k v_l. k∑​uk​⋅l∑​vl​=k,l∑​uk​vl​.

这里的乘积是一个二重级数

u0v0+u0v1+u0v2+u0v3+⋯+u1v0+u1v1+u1v2+u1v3+⋯+u2v0+u2v1+u2v2+u2v3+⋯+⋯\begin{aligned} & u_0 v_0 + u_0 v_1 + u_0 v_2 + u_0 v_3 + \cdots \\\\ & + u_1 v_0 + u_1 v_1 + u_1 v_2 + u_1 v_3 + \cdots \\\\ & + u_2 v_0 + u_2 v_1 + u_2 v_2 + u_2 v_3 + \cdots \\\\ & + \cdots \end{aligned} ​u0​v0​+u0​v1​+u0​v2​+u0​v3​+⋯+u1​v0​+u1​v1​+u1​v2​+u1​v3​+⋯+u2​v0​+u2​v1​+u2​v2​+u2​v3​+⋯+⋯​

绝对收敛性意味着可按任意顺序求和，其值不变。例如可按 k+l=nk + l = nk+l=n 的大小顺序排列，

∑k=0∞uk⋅∑l=0∞vl=∑n=0∞wn,wn=∑k=0nukvn−k.\sum_{k=0}^{\infty} u_k \cdot \sum_{l=0}^{\infty} v_l = \sum_{n=0}^{\infty} w_n, \quad w_n = \sum_{k=0}^{n} u_k v_{n-k}. k=0∑∞​uk​⋅l=0∑∞​vl​=n=0∑∞​wn​,wn​=k=0∑n​uk​vn−k​.

而且如果限于这种求和次序，则乘法的条件还可以放宽：∑uk\displaystyle\sum u_k∑uk​，∑vl\displaystyle\sum v_l∑vl​ 都收敛，且其中之一绝对收敛；或 ∑uk\displaystyle\sum u_k∑uk​，∑vl\displaystyle\sum v_l∑vl​ 和 ∑wn\displaystyle\sum w_n∑wn​ 都收敛。

· 二重级数

所谓二重级数，指的是排列成

a11+a12+a13+a14+⋯+a1n+⋯+a21+a22+a23+a24+⋯+a2n+⋯+⋯+am1+am2+am3+am4+⋯+amn+⋯+⋯\begin{aligned} & a_{11} + a_{12} + a_{13} + a_{14} + \cdots + a_{1n} + \cdots \\\\ & + a_{21} + a_{22} + a_{23} + a_{24} + \cdots + a_{2n} + \cdots \\\\ & + \cdots \\\\ & + a_{m1} + a_{m2} + a_{m3} + a_{m4} + \cdots + a_{mn} + \cdots \\\\ & + \cdots \end{aligned} ​a11​+a12​+a13​+a14​+⋯+a1n​+⋯+a21​+a22​+a23​+a24​+⋯+a2n​+⋯+⋯+am1​+am2​+am3​+am4​+⋯+amn​+⋯+⋯​

的方阵，这个方阵的右端和下端都是无限的。方阵的每一项用 akla_{kl}akl​ 表示，其中的第一个指标 kkk 表示行，第二个指标 lll 表示列。

现在求出方阵的前 mmm 行 nnn 列共 m×nm \times nm×n 项之和

Smn=∑1≤k≤m1≤l≤nakl,S_{mn} = \sum_{\substack{1 \leq k \leq m \\ 1 \leq l \leq n}} a_{kl}, Smn​=1≤k≤m1≤l≤n​∑​akl​,

根据 SmnS_{mn}Smn​ 就能构造出这个二重级数的部分和序列。容易理解，如果部分和序列收敛，

lim⁡m→∞n→∞Smn=S,\lim_{\substack{m \to \infty \\ n \to \infty}} S_{mn} = S, m→∞n→∞​lim​Smn​=S,

则称此二重级数收敛。SSS 就是这个二重级数之和：

S=∑k,l=1∞akl.S = \sum_{k,l=1}^{\infty} a_{kl}. S=k,l=1∑∞​akl​.

这时，∀ε>0\forall \varepsilon > 0∀ε>0，总可以找到正整数 NNN，当 m,n>Nm,n > Nm,n>N 时，恒有

∣Smn−S∣<ε.|S_{mn} - S| < \varepsilon. ∣Smn​−S∣<ε.

除了这种求和方式之外，当然还可以考虑其他求和方式。例如，考虑到上面的部分和

Smn=∑1≤k≤m1≤l≤nakl=∑k=1m(∑l=1nakl)=∑l=1n(∑k=1makl),S_{mn} = \sum_{\substack{1 \leq k \leq m \\ 1 \leq l \leq n}} a_{kl} = \sum_{k=1}^{m} \left( \sum_{l=1}^{n} a_{kl} \right) = \sum_{l=1}^{n} \left( \sum_{k=1}^{m} a_{kl} \right), Smn​=1≤k≤m1≤l≤n​∑​akl​=k=1∑m​(l=1∑n​akl​)=l=1∑n​(k=1∑m​akl​),

还可以有累次求和：

∑k=1∞(∑l=1∞akl)=lim⁡m→∞(lim⁡n→∞Smn)or∑l=1∞(∑k=1∞akl)=lim⁡n→∞(lim⁡m→∞Snm).\sum_{k=1}^{\infty} \left( \sum_{l=1}^{\infty} a_{kl} \right) = \lim_{m \to \infty} \left( \lim_{n \to \infty} S_{mn} \right) \quad \text{or} \quad \sum_{l=1}^{\infty} \left( \sum_{k=1}^{\infty} a_{kl} \right) = \lim_{n \to \infty} \left( \lim_{m \to \infty} S_{nm} \right). k=1∑∞​(l=1∑∞​akl​)=m→∞lim​(n→∞lim​Smn​)orl=1∑∞​(k=1∑∞​akl​)=n→∞lim​(m→∞lim​Snm​).

前者是先按列求和，再将各列之和相加（可称为逐列求和）；后者的求和次序则相反（可称为逐行求和）。

需要注意，即使二重级数收敛，某些行或列的和也不一定存在，因此累次求和的和也不一定存在。

而且，如果逐行和逐列求和的和都存在，这两个和数也不一定相等（即和数与求和次序有关）。

二重级数的和是否依赖于求和方式，原则上与级数是否绝对收敛有关。如果二重级数绝对收敛，则级数各项的先后次序可以重新排列，因而不同求和方式得到相同的和数。

作为特殊情况，收敛半径可以是 0，即收敛圆退化为一个点。除 z=az = az=a 点外，幂级数在全复平面处处发散；也可以是 ∞\infty∞，收敛圆就是全复平面。幂级数在全复平面收敛，但在 ∞\infty∞ 点肯定发散，除非此幂级数只有常数项一项。

幂级数在收敛圆内一定收敛，在收敛圆外一定发散。在收敛圆的圆周上，级数可能在所有点都收敛，可能在所有点都发散，也可能在一部分点收敛，在另一部分点发散。例如：

但不论哪种情况，幂级数的收敛圆的圆周上总肯定有和函数的奇点。特别需要说明，即使在和函数的奇点处，幂级数仍然可能收敛。读者可以求出上面三个级数的和函数，验证它们在 z=1z = 1z=1 点不解析。

当幂级数 ∑n=0∞cn(z−a)n\displaystyle\sum_{n=0}^{\infty} c_n(z-a)^nn=0∑∞​cn​(z−a)n 在收敛圆周上某点 z0z_0z0​ 收敛时，其和与级数在收敛圆内的和函数之间的关系，有下面的定理。

定理2 (Abel 第二定理)

若幂级数 ∑n=0∞cn(z−a)n\displaystyle\sum_{n=0}^{\infty} c_n(z-a)^nn=0∑∞​cn​(z−a)n 在收敛圆内收敛到 f(z)f(z)f(z)，且在收敛圆周上某点 z0z_0z0​ 也收敛，和为 S(z0)S(z_0)S(z0​)，则当 zzz 由收敛圆内趋于 z0z_0z0​ 时，只要保持在以 z0z_0z0​ 为顶点、张角为 2ϕ<π2\phi < \pi2ϕ<π 的范围内，f(z)f(z)f(z) 就一定趋于 S(z0)S(z_0)S(z0​)。

需要明确一下（当收敛半径为有限值时）“和函数的奇点（或解析性）”这种说法的含义。毫无疑问，幂级数在收敛圆内收敛，在收敛圆外发散，因而幂级数只在收敛圆内有定义，并且代表了一个解析函数。另一方面，就和函数而言，尽管幂级数是在收敛圆内才收敛到和函数，但是，我们从来不会认为这个函数只是局限于收敛圆内才有定义（少数函数除外）。

事实上，幂级数只不过是这个函数在收敛圆内的一种表达形式，在其他区域内可以有其他表达形式。我们总会自觉或不自觉地在整个定义域上来考察它的解析性。我们也正是在这个前提下，才能谈论函数在收敛圆周上乃至收敛圆外的奇点。
